# Code of Conduct

All contributors and users of the Nerdolando.Bff project are expected to help keep a **friendly, respectful and professional** environment.

## Our standards

Examples of behavior that contributes to a positive environment:

- Being respectful and considerate in discussions.
- Giving and gracefully accepting constructive feedback.
- Focusing on what is best for the project and its users.
- Using welcoming and inclusive language.

Examples of unacceptable behavior:

- Insults, personal or political attacks.
- Harassment, trolling, or derogatory comments.
- Public or private harassment of any contributor or user.
- Publishing others’ private information.

## Scope

This Code of Conduct applies within all project spaces, including:

- Issues, pull requests, and code review comments,
- Discussions,
- Any other channel related to this project.

## Enforcement

If you experience or witness behavior that violates this Code of Conduct, please report it privately to the maintainer:

- E-mail: `admin@nerdolando.pl`

All reports will be reviewed in good faith.  
The maintainer may take any action they deem appropriate, for example:

- asking the person to stop the behavior,
- removing comments or closing discussions,
- temporarily or permanently restricting participation in the project.
